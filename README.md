# Java MCA Assignments (Sem-2)

This repository contains **all 7 assignments for Java (MCA – Semester 2)**, organized folder-wise.

## Repository Structure

- `Assignment_1/`
- `Assignment_2/`
- `Assignment_3/`
- `Assignment_4/`
- `Assignment_5/`
- `Assignment_6/`
- `Assignment_7/`

Each folder contains the Java source code and related files for that assignment.

## How to Run

1. Open the required `Assignment_*` folder.
2. Compile the Java file(s):

   ```bash
   javac FileName.java
   ```

3. Run the compiled program:

   ```bash
   java FileName
   ```

> Tip: If you are using an IDE (IntelliJ IDEA / Eclipse / VS Code), you can import the folder as a project and run files directly.

## Requirements

- Java JDK 8+ (recommended: latest LTS)
